# ?? Быстрый старт - AI Assistant Extension для VS

## ? За 5 минут

### Шаг 1: Запуск расширения
1. Откройте проект в Visual Studio
2. Нажмите **F5** для запуска в режиме отладки
3. Откроется экспериментальный экземпляр VS

### Шаг 2: Открыть меню расширения
- Перейдите в **Tools ? AI Assistant**
- Вы увидите две команды:
  - **Open AI Chat** ??
  - **Settings** ??

### Шаг 3: Настроить подключение
1. Нажмите **Tools ? AI Assistant ? Settings**
2. Заполните поля:
   - **URL сервера**: `http://localhost:8000` (или ваш адрес)
   - **API ключ**: `your-api-key-here`
   - **Таймаут**: 30000
   - **SSL/TLS**: Включено (если используется HTTPS)
3. Нажмите **Проверить соединение** ?
4. Нажмите **Сохранить**

### Шаг 4: Открыть чат
1. Перейдите в **Tools ? AI Assistant ? Open AI Chat**
2. Внизу VS появится окно чата
3. Введите вопрос и нажмите **Enter** или **Отправить**

Все настройки автоматически сохраняются:
```
%AppData%\AIAssistant\settings.json
C:\Users\<Your Username>\AppData\Roaming\AIAssistant\settings.json
```

## ??? Что делать, если что-то не работает?

### Проблема: "Расширение не загружается"
**Решение:**
1. Убедитесь, что решение собирается: `Ctrl + Shift + B`
2. Перестройте решение: `Build ? Rebuild Solution`
3. Очистите: `Build ? Clean Solution`
4. Попробуйте снова: `F5`

### Проблема: "Окно чата не открывается"
**Решение:**
1. Проверьте вывод отладки: `View ? Output (Ctrl + Alt + O)`
2. Посмотрите логи
3. Перезапустите VS

### Проблема: "Не могу подключиться к серверу"
**Решение:**
1. Откройте Settings: `Tools ? AI Assistant ? Settings`
2. Проверьте URL сервера
3. Проверьте API ключ
4. Нажмите "Проверить соединение"
5. Убедитесь, что ваш сервер запущен
