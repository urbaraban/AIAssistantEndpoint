# ?? Пошаговая инструкция - AI Assistant Extension

## Шаг 1: Запуск расширения

1. Откройте проект **AIAssistantEndpoint** в Visual Studio
2. Убедитесь, что проект собирается без ошибок (Ctrl + Shift + B)
3. Нажмите **F5** или **Debug ? Start Debugging**

```
?

Там будет установлено наше расширение
```

После запуска экспериментального VS:

1. Поднимитесь в верхнюю часть окна VS

3. Нажмите на **Tools**

## Шаг 4: Ввести параметры подключения

1. **URL сервера** - введите адрес вашего AI сервера
   - Пример: `http://localhost:8000`
   - Или: `https://your-server.com/api`

2. **API ключ** - введите ваш ключ доступа
   - Символы скрыты звёздочками (*)

3. **SSL/TLS** - включите если используется HTTPS
   - По умолчанию: ? (включено)
   - Отключите галочку если используется HTTP

4. **Таймаут** - время ожидания ответа в миллисекундах
   - По умолчанию: 30000 (30 секунд)
   - Уменьшите если сервер быстрый
   - Увеличьте если сервер медленный

## ?? Если что-то не работает

### "Окно чата не открывается"
1. Проверьте логи: View ? Output
2. Убедитесь что решение собирается
3. Перезагрузите VS

### "Не подключается к серверу"
1. Откройте Settings (??)
2. Проверьте URL и API ключ
3. Нажмите "Проверить соединение"
4. Убедитесь что ваш сервер работает

### "Расширение не видно в меню"
1. Перестройте решение: Ctrl + Shift + B
2. Очистите: Build ? Clean Solution
3. Запустите заново: F5
